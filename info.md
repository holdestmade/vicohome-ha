# VicoHome for Home Assistant

**Custom integration for VicoHome CG1 cameras**

- 📹 Motion detection via cloud events
- 📸 Snapshot camera entity
- 📊 Event sensors
- 🔔 Telegram notifications (photo + video)
- ⚙️ Runtime config via HA UI

**Unofficial** – reverse-engineered cloud API. Tested with **CG1** model only.
